To execute the solution:
1. Install Python-3 (https://www.python.org/downloads/)
2. Install all dependencies listed in the requirements.txt file, by running "pip install -r requirements.txt"
3. Make migrations by running "python manage.py makemigrations"
4. Apply migrations by running "python manage.py migrate"
5. Collect static files by running "python manage.py collectstatic"
5. Execute the project by running "python manage.py runserver"
